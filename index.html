Série de Taylor

A série de Taylor é uma representação de uma função como uma soma infinita de termos calculados a partir dos valores das derivadas da função em um único ponto. Aqui, aproximamos a função ln(x) em torno de x = 1.

4
0.5-2.0
// Código Python para Série de Taylor def taylor_ln(x, n): terms = [((-1)**(k+1)) * ((x-1)**k) / k for k in range(1, n+1)] return sum(terms)

Série de Maclaurin

A série de Maclaurin é um caso especial da série de Taylor, centrada em zero. Aqui, aproximamos a função seno usando a série de Maclaurin.

3
-10-10
// Código Python para Série de Maclaurin def maclaurin_sin(x, n): result = sum(((-1)**k) * (x**(2*k+1)) / math.factorial(2*k+1) for k in range(n)) return result

Série de Fourier

A série de Fourier representa uma função periódica como uma soma de senos e cossenos. Aqui, aproximamos a função f(x)=x usando a série de Fourier.

10
// Código Python para Série de Fourier def fourier_series(x, n_terms): result = 0 for n in range(1, n_terms+1): result += 2*((-1)**(n+1) / n) * math.sin(n*x) return result
